## ✨ About The Project

**TextModPro** is a lightweight web tools platform designed to make everyday digital tasks easier and faster.  
It provides useful utilities like text conversion, QR code generation, password creation, and more—all in one place.

Built with simplicity, speed, and user experience in mind.

- 🔤 Case Converter (Uppercase, lowercase, etc.)
- 🔢 QR Code Generator
